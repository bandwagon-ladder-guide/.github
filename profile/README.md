## 搭梯子应该选什么协议？

这里简单做个对比，帮你理清思路：

**Shadowsocks（SS/SS-libev）**：配置简单，兼容性好，适合入门。采用 AEAD 加密（如 chacha20-ietf-poly1305）时，安全性很高，且对重放攻击有抵御能力。

**V2Ray / Xray**：功能更全面，支持多种传输协议。VLESS+XTLS 方案将流量伪装成正常 HTTPS，隐蔽性极强，是目前隐私保护效果最好的方案之一。

**Trojan**：原理是把流量伪装成合法的 HTTPS 网站访问，检测难度高。适合对隐蔽性要求高的用户。

对于刚入手搬瓦工的新用户，建议从 Shadowsocks-libev 开始搭，配置简单，15 分钟能搞定；有一定基础的话，直接上 Xray（VLESS+XTLS），安全和速度兼顾。
